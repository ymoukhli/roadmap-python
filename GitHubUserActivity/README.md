# GitHub User Activity

A CLI tool that fetches and displays GitHub user activity using the GitHub API.

## Features

- View user profile details (followers, following count)
- List followers and users being followed
- Browse recent events (pushes, pull requests, issues, forks, etc.)
- Interactive menu-driven navigation

## Usage

```bash
python3 main.py
```

You will be prompted to enter a GitHub username. From there, use the menu to explore:

1. **Followers** - List the user's followers
2. **Following** - List users they follow
3. **Events** - View recent activity (push, PR, issue, fork, etc.)
4. **Exit** - Quit the program

## Requirements

- Python 3.x
- No external dependencies (uses only `urllib` and `json` from the standard library)
